# Contributing to AgentAuth

First off, thank you for your interest in contributing to AgentAuth.

AgentAuth is an authentication and credential broker for AI agents. Contributions are welcome, especially around security, integrations, and performance.

## License

The core AgentAuth server is licensed under the GNU Affero General Public License v3.0 (AGPL-3.0). By contributing, you agree that your contributions will be licensed under AGPL-3.0.

In addition, substantial contributions require signing the AgentAuth Contributor License Agreement (CLA), which grants the project maintainer additional rights described in `CLA.md`.

We cannot merge non-trivial contributions unless the CLA has been accepted.

## How to contribute

1. **Open an issue** describing the bug, feature, or refactor you are proposing.
2. **Discuss** the approach briefly in the issue, if it's a larger change.
3. **Fork** the repository and create a topic branch from `main`.
4. **Submit a pull request** referencing the issue.

If this is your first substantial contribution, you will be asked to confirm that you accept the CLA. See `CLA.md` for details.

## Code guidelines

- Write clear, self-explanatory code and tests.
- Add or update tests for any behavior change.
- Keep commits focused; small, cohesive changes are easier to review.

Thank you again for helping improve AgentAuth.
